# Job Application Tracker

A modern, beautiful web app to help you track your job applications easily!

## Features

- Gorgeous glassmorphism UI and colorful gradient background
- Emoji & color-coded status badges (Applied, Interview, Offer, Rejected)
- Responsive design for desktop & mobile
- Add company, position, application date, status, and notes
- Animated feedback cards and “toast” notifications

## How To Use

1. Fill out the form with your job application details.
2. Click “Add Application.”
3. View all your applications listed in beautiful cards below.
